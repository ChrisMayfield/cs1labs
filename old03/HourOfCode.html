CS 139: Programming Fundamentals
James Madison University, Spring 2014 Semester

Lab01A: Hour of Code tutorial

Background

In December 2013 during CSEdWeek, over fifteen million students had the opportunity to try out programming for an hour. This unprecedented campaign, sponsored by Code.org and a large number of supporting organizations, become the fastest-spreading technology in history! Today you will work through one or more Hour of Code tutorials and make connections to the "fundamentals" that we discussed in class.

Objectives

  • Complete all 20 puzzles of the "Write your first computer program" tutorial.

  • Explain how repeat-loops, conditionals, and basic algorithms work together.

  • Understand the relationship between blocks and underlying JavaScript code.

Instructions

  1. Open the official Code.org tutorial. The lab machines have built-in speakers, but you may want to plug in headphones.

  2. Download the Lab01A.txt worksheet and open/edit the file using a text editor. (In the Linux lab, use gedit or geany.)

  3. Submit your completed worksheet as a plain text file via canvas.jmu.edu by the end of the lab period.

  4. Feel free to explore other Hour of Code tutorials. There are dozens to choose from, and they're all great!
